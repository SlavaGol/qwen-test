## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd qwen-test
   ```

2. Установите необходимые зависимости (если применимо):
   ```bash
   # Пример для Python
   pip install -r requirements.txt

   # Пример для Node.js
   npm install
   ```
